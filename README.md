## About The Project

포스트 발행, 읽기, 로그인, 로그아웃 등의 기능이 구현된 아주 간단한 블로그 웹사이트입니다. 

### Features

현재 가능한 페이지: 

- Home, Routing`/`
- Contact, Routing`/contact`
- Compose, route`/compose`
- Login, route`/login`
- Logout, route`/logout`
- Sign up, route`/signUp`
- Specific Post, route`/posts/:postID`
